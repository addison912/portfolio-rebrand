Learnify

Summary

Learnify was built over the course of a week as an excercise in Django and Python for the Web Developement Immersive at General Assembly.

Users can create a profile, create courses and upload course videos. Users can also purchase courses using the Stripe API for payments.

This site is currently hosted on Heroku, (as per the requirements for the assignment). Uploded photos and videos will only remain on the site for an hour.
